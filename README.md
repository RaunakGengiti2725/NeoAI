# NeoAI
NeuroBridgeAI

## Running the backend

```bash
cd backend
pip install -r requirements.txt
python app.py
```

The API server listens on `http://localhost:5000` by default.

## Running the frontend

```bash
cd frontend
npm install
npm run dev
```

Set `VITE_API_URL` to the backend URL if you serve them separately.
